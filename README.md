# Backdrop Wallpaper App
This app is a cross-platform wallpaper discovery and personalization system built using the Unsplash API. Users can register, log in, explore a vast collection of high-quality wallpapers, and set their favorites directly as their phone background. The app includes a Flutter mobile client and a Node.js backend for authentication and data storage.
